### description
<br />
Clean and simple analogue watchface, includes:
 - day of the week
 - date of the month
 - battery percentage
 - steps data
 - sleep data
 - bluetooth symbol<br />
<br />

Options:
 - show or hide date
 - show or hide health
 - show or hide battery
